# UPI-Transactions
Power BI dashboard analyzing UPI transaction data with interactive filters and bookmark-based visual switching to explore digital payment trends.

# 💳 UPI Transactions Analytics Dashboard

## 📌 Project Overview
This project analyzes **UPI (Unified Payments Interface) transaction data** using **Power BI** to uncover insights about digital payment trends, transaction patterns, and banking activity.

The dashboard provides an **interactive environment** where users can explore transaction data across multiple dimensions such as **banks, cities, device types, and demographics**.

A key feature of this dashboard is the use of **Power BI Bookmarks**, enabling users to switch between different visual representations of the same data.

---

## 📊 Dashboard Features

The dashboard includes multiple interactive visualizations to analyze transaction data effectively.

Key insights include:

- 📈 Monthly transaction trend analysis  
- 🏦 Bank-wise transaction insights  
- 🌆 City-wise transaction comparison  
- 📱 Device type usage analysis  
- 👤 Gender and age group analysis  
- 💳 Payment method breakdown  
- 💰 Transaction amount and remaining balance tracking  

Interactive **slicers** allow filtering by:

- Bank Name Sent  
- Bank Name Received  
- City  
- Device Type  
- Gender  
- Age Groups  
- Merchant Name  
- Payment Method  
- Purpose  
- Transaction Type  

---

## 🔖 Bookmark Feature

This dashboard utilizes **Power BI Bookmarks** to dynamically change the visualization type without navigating to a different page.

Available bookmark views include:

- Column Balance
- Line Balance
- Column Spending
- Line Remaining Balance

This improves dashboard usability by allowing users to quickly explore different perspectives of the transaction data.
